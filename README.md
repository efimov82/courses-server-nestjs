# course-server

## Description

REST API Courses server on Nodejs + NestJs

## Installation

Copy file .env.example to .env and edit settings

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Docker

```bash
# start container
$ sudo docker-compose up

Swagger API doc:
Open http://localhost:3000/api

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
